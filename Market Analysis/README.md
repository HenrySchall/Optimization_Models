# Examples

Example 1 

|Produto          |Lucro Unitário|Variável|Valor    |
|---	          |---            |---    |---      |
|A                |12             |X1     | Y       | -> 80
|B                |18             |X2     | Y       | -> 77
|C                |22             |X3     | Y       | -> 0

|Produto          |Máquina 1|Máquina 2|Máquina 3 |
|---	          |---      |---    |---         |
|A                |1,5      |0      |1,2         |
|B                |0        |2,2    |2           |
|C                |1,2      |1,4    |2,4         |
|Tempo Disponível |120      |200    |250         |
|Tempo de Uso     |0        |0      |0           |

Maximizar -> 12X1 + 18X2 + 22X3

Sujeito:
- 1,5X1 + 1,2X3 ≤ 120
- 2,2X2 + 1,4X3 ≤ 200
- 1,2X1 + 2X2 + 2,4X3 ≤ 120
Sendo que: X1, X2, X3 ≥ 0
